This Python project checks whether a specific port on a given host or IP address is open or closed.

## Features

* Accepts host/IP address as input
* Accepts port number as input
* Checks port status using Python sockets
* Displays whether the port is OPEN or CLOSED

## Technologies Used

* Python 3
* Socket Module

## How to Run

1. Clone the repository
2. Open terminal in the project folder
3. Run:

```bash
python port_status_checker.py
```

or

```bash
py port_status_checker.py
```

## Example

Enter Host/IP Address: google.com

Enter Port Number: 80

Port 80 is OPEN on google.com
